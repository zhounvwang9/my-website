# Northern Sky Crystals - 水晶首饰网站

一个适配移动端的水晶首饰展示网站，支持中英文双语切换。

## 项目特点

### 设计风格
- 🎨 温暖优雅的大地色系（米色、粉色、棕色）
- 📱 移动端优先的响应式设计
- ✨ 简约现代的用户界面
- 🌟 流畅的动画效果

### 核心功能
- 🌐 中英文双语切换
- 📦 产品分类筛选
- 🛍️ 产品详情展示
- 🎯 侧边导航菜单
- 💫 平滑滚动效果
- 🔝 返回顶部按钮

### 页面结构
1. **首页 (index.html)**
   - 品牌横幅
   - 产品分类导航
   - 促销活动展示
   - 精选产品展示
   - 品牌介绍

2. **产品列表页 (products.html)**
   - 分类筛选功能
   - 产品网格展示
   - 16+ 产品展示

3. **产品详情页 (product-detail.html)**
   - 产品图片画廊
   - 尺寸和材质选择
   - 详细产品描述
   - 搭配推荐
   - 相关产品推荐

4. **关于我们页 (about.html)**
   - 品牌故事
   - 核心价值观
   - 制作工艺流程
   - 联系信息

## 使用方法

### 本地运行
1. 下载所有文件到本地目录
2. 用浏览器打开 `index.html` 文件
3. 或使用本地服务器运行（推荐）

### 使用本地服务器（推荐）
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 功能说明

### 语言切换
- 点击右上角的语言切换按钮（EN/中文）
- 语言偏好会保存在浏览器本地存储中
- 页面所有文本会自动切换

### 产品筛选
- 在产品列表页点击分类按钮
- 支持的分类：全部、项链、手链、水晶、水晶灯
- 筛选结果会带有淡入动画

### 侧边菜单
- 点击右上角的菜单图标打开
- 点击关闭按钮或菜单外部关闭
- 点击菜单项后自动关闭

### 产品详情
- 点击图片缩略图切换主图
- 选择尺寸和材质选项
- 加入购物车会显示提示信息

## 自定义配置

### 修改颜色主题
在 `css/style.css` 文件的 `:root` 部分修改 CSS 变量：

```css
:root {
    --primary-color: #d4a574;      /* 主色调 */
    --secondary-color: #f5e6d3;    /* 次要色 */
    --text-dark: #2c2c2c;          /* 深色文字 */
    --text-light: #666;            /* 浅色文字 */
}
```

### 添加产品
在相应的 HTML 文件中复制产品卡片代码块，修改以下内容：
- 产品ID (`href="product-detail.html?id=X"`)
- 产品名称（中英文）
- 价格
- 背景渐变色
- 图标/emoji

### 修改文本内容
所有需要双语的文本都使用 `data-zh` 和 `data-en` 属性：

```html
<h1 data-zh="中文文本" data-en="English Text">中文文本</h1>
```
